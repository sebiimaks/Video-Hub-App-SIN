A fork of the great Video Hub App to add some things I need. All done with an LLM, so not prepared to stand behind it, although it works well for me. 

All changes made as of 18/07/2026:

  - Added JSON editor to be able to manually fix verious issues with the main release. Easier than debugging and fixing the individual issues.
  - Added option to reset 'times played'.
  - Added timeout fixes that were causing thumbnail generation problems with some files, especially on network drives.
  - Added confirmation dialog box after delete selection from context menu.
  - Fixed lint checks to use ESLint instead of the removed TSLint tool.

Big thanks to [whyboris](https://github.com/whyboris) for building this great app.
